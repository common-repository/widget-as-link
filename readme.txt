Adds possibility to create clickable widget for external and internal links. Support Widget Clone, Widget Background, user CSS Classes, Auxiliary color mark and Quick show/hide options!

== Description ==

This plugins adds an extra fields to all widgets that lets you add a Link, Background, User css, Auxiliary color mark, quick Show/Hide & Clone to the widget's. When the widget is displayed, the widget is now clickable to the url you entered.

Works with all widgets!

= Options =
* Open links in new tab/window
* Wrap entire widget in the link. Useful when you need to make the whole widget clickable, not just the title text or description.

== Installation ==

1. Upload `wp-widget-as-link` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I link the widget to current page or post? =
You can use the `wp-widget-as-link` for add link whatever you can imagine.
